                                            # 文档简介

                                            本文档主要描述迅捷到家系统的功能需求点及其设计，目的在于清晰地定义各页面的需求细节及逻辑流程。

                                            # 文档读者

                                            本文档主要面向以下读者：SP2 项目的研发人员、测试人员、产品经理、市场运营人员、管理人员以及教学老师、学生等。

                                            # 项目简介

                                            # 定位

                                            【迅捷到家】是普通用户通过购物平台进行购买的在线系统。

                                            # 登录模块

                                            • 用户注册

                                              用户注册功能，注册账号后方可登录进入首页。

                                            • 用户登录

                                              已注册的用户可以在登录页面登录进入首页。

                                            • 登录验证

                                              只有登录账号才能进入首页使用应用。

                                            # 商品模块（首页）

                                            • 附近的商店

                                              已登录用户可以查看附近的商店信息。

                                            • 进入商店

                                              已登录的用户可以进入某个商店，查看商店的所有商品信息。

                                            • 商店详情

                                              已登录的用户可以进入某个商店，查看商店的所有商品信息和商店信息。

                                            • 热门商品

                                              已登录的用户可以查看商品的详细信息。

                                            • 商品分类

                                              用户可以根据商品类型进行查看商品的信息。

                                            • 搜索商品

                                              用户可以根据

                                            • 查询关注

                                              已登录的用户可以查看曾经关注的商店。

                                            • 修改关注

                                              已登录的用户可以取消已关注的商店。

                                            # 订单模块

                                            • 收货地址

                                              用户可以对收货地址进行管理。

                                            • 订单查询

                                              用户可以查看未完成、待发货、待收货等订单。

                                            • 订单生成

                                              加入购物车的商品可以直接生成订单。

                                            # 购物车模块

                                            • 查看购物车

                                              查看已选的购物车，并可以按商店进行结算。

                                            # 用户模块

                                            • 查看用户信息

                                              用户可以看到自己的姓名和 ID

                                            • 管理地址

                                              用户可以对自己的收货地址进行管理
