                              # 支付模块

                              # 1-开发者注册

                              # 文档介绍

                              根据订单信息完成支付功能。调用支付宝接口进行支付。支付宝开放平台提供了沙箱支付环境。沙箱支付环境和正式支付操作流程是一样的，只是不会产生真实的费用。为了方便学习，这里采用沙箱环境进行操作。要想使用沙箱环境，需要入住支付宝平台成为开发者。下面我们来介绍一下如何成为开发者，并获取沙箱环境的信息。

                              # 沙箱环境

                              # 进入官网

                              先进入支付宝开放平台官网

                              # 扫码登录

                              再用支付宝账号登录，登录开放平台后，若未完成入驻，则系统自动引导进入入驻流程。入驻完毕选择研发服务。

                              # 沙箱应用

                              从研发服务进入沙箱环境。我们在沙箱环境里可以看到应用归属商户 UID。这里一定要使用自己注册的。还能看到支付宝网关：https://openapi.alipaydev.com/gateway.do。为了操作更加方便这里我们选择系统默认密钥，开启一下 RSA2 密钥（公钥模式)。

                              我们点击一下查看。可以看到应用密钥和应用公钥以及支付宝公钥。

                              # 沙箱账号

                              沙箱账号，一个为商家信息，一个为买家信息。

                              # 沙箱钱包

                              如果是手机扫码支付，需要下载手机版沙箱钱包。同学们也可以不下载沙箱钱包，直接选择网页支付进行测试。

                              上面沙箱环境所提供的信息，是为后面的编码做准备。接着我们继续完成下一节实验【支付与退款】，如何在代码里面使用这些信息进行支付和退款。

                              # 2-支付与退款

                              根据订单信息完成支付功能。调用支付宝接口进行支付。支付宝开放平台提供了沙箱支付环境。沙箱支付环境和正式支付操作流程是一样的，只是不会产生真实的费用。为了方便学习，本实验采用沙箱支付环境进行操作。

                              # 添加配置

                              # 添加支付依赖

                              在使用支付宝支付时，需要导入支付 API 相关 Jar。同学们只需要把如下代码复制到工程 pom.xml 里即可。

                              <!--支付 API-->
<dependency>
    <groupId>com.alipay.sdk</groupId>
    <artifactId>alipay-sdk-java</artifactId>
    <version>3.3.4.ALL</version>
</dependency>

                              # 添加属性文件

                              支付宝支付时，需要配置一下开发者入住时的基本信息。我们把它放到一个属性文件里，方便管理。

                              paymentconfig.properties

                              创建属性文件

                              touch  src/main/resources/paymentconfig.properties

                              下面为属性文件模板，同学们需要把部分等于号右边的值替换成自己注册时的信息。

                              APPID=你的 APPID
RSA_PRIVATE_KEY=你的私钥 pkcs8 格式的
## 请求网关地址
URL=https://openapi.alipaydev.com/gateway.do
##编码
CHARSET=UTF-8
##返回格式
FORMAT=json
##支付宝公钥
ALIPAY_PUBLIC_KEY=
#记录日志目录
log_path=/tmp/log
##RSA2
SIGNTYPE=RSA2
##服务器异步通知页面路径
notify_url=xxx/payresult
##页面跳转同步通知页面路径
return_url=xxx/payresult

                              接着把属性文件纳入 spring 管理，参考配置如下：

                              <bean class="org.springframework.beans.factory.config.PropertyPlaceholderConfigurer">
    <property name="ignoreUnresolvablePlaceholders" value="true" />
    <property name="locations">
        <list>
            <!--加载支付的属性文件-->
            <value>classpath:paymentconfig.properties</value>
            <!--我们也可以把 JDBC 的属性文件也移动到这里-->
            <value>classpath:jdbc.properties</value>
        </list>
    </property>
</bean>

                              # 加载属性文件

                              先创建一个用于加载属性文件 paymentconfig.properties 的类。

                              • AlipayConfig.java

                                创建 Java 类 AlipayConfig.java。

                                mkdir  src/main/java/com/config
touch  src/main/java/com/config/AlipayConfig.java

                                在 AlipayConfig.java 类里编写如下内容。

                                package com.config;
import org.springframework.beans.factory.annotation.Value;
public class AlipayConfig {
    // 商户appid
    @Value("${APPID}")
    public  String APPID;
    // 私钥 pkcs8格式的
    @Value("${RSA_PRIVATE_KEY}")
    public  String RSA_PRIVATE_KEY;
    // 服务器异步通知页面路径
    @Value("${notify_url}")
    public  String notify_url;
    // 页面跳转同步通知页面路径
    @Value("${return_url}")
    public  String return_url;
    // 请求网关地址
    @Value("${URL}")
    public  String URL;
    // 编码
    @Value("${CHARSET}")
    public  String CHARSET;
    // 返回格式
    @Value("${FORMAT}")
    public  String FORMAT;
    // 支付宝公钥
    @Value("${ALIPAY_PUBLIC_KEY}")
    public  String ALIPAY_PUBLIC_KEY;
    // 日志记录目录
    @Value("${log_path}")
    public  String log_path;
    // RSA2
    @Value("${SIGNTYPE}")
    public  String SIGNTYPE;
}

                                将 AlipayConfig.java 纳入 Spring 管理。

                                在 applicationContext.xml 文件里添加如下配置。

                                <bean class="com.config.AlipayConfig"></bean>

                              # IOrderMapper.java

                              里面需要提供两个方法。

                              1. updateOrder 用于支付成功时修改订单状态。
                              2. getTotalPriceByOrdernumber 退款时用于查询退款的金额。
                              public interface IOrderMapper {
    
    @Select("select orderId,addressId,userId,updatedAt,createdAt,shopId,orderTotalPrice," +
            "isCanceled,ordernumber from t_order where orderId = #{orderId} and userId =" +
            " #{userId} and isCanceled = #{isCanceled}")
   
    Order getOrder(
            @Param("userId")
                    Long userId,
            @Param("orderId")
                    Long orderId,
            @Param("isCanceled")
                    String isCanceled);
    
    @Update("update t_order set isCanceled=#{isCanceled} where ordernumber=#{ordernumber}")
    Boolean updateOrder(@Param("ordernumber")
                                String out_trade_no,
                        @Param("isCanceled")
                                String isCanceled);

    @Select("select orderTotalPrice from t_order where ordernumber = #{ordernumber}")
    String getTotalPriceByOrdernumber(@Param("ordernumber") String out_trade_no);
}

                              # PayService.java

                              创建一个用于支付的服务类。

                              touch  src/main/java/com/service/PayService.java

                              支付服务类，里面提供三个方法，支付（payment）、退款（refund）和修改订单状态（updateOrder）方法。

                              @Service
public class PayService {
    @Autowired
    IOrderMapper orderMapper;
    @Autowired
    AlipayConfig alipayConfig;
    //支付方法
    public String payment(Long userId,Long orderId) {

        Order order =  orderMapper.getOrder(userId,orderId,"1");
        String form = "";
        // 商户订单号，商户网站订单系统中唯一订单号，必填
        String out_trade_no = order.getOrdernumber();
        // 订单名称，必填
        String subject = order.getOrdernumber();
        // 付款金额，必填
        String total_amount=order.getOrderTotalPrice().toString();

        // 超时时间 可空
        String timeout_express="2m";
        // 销售产品码 必填
        String product_code=order.getOrderId().toString();
        /**********************/
        // SDK 公共请求类，包含公共请求参数，以及封装了签名与验签，开发者无需关注签名与验签
        //调用RSA签名方式
        AlipayClient client = new DefaultAlipayClient(alipayConfig.URL, alipayConfig.APPID,
                alipayConfig.RSA_PRIVATE_KEY, alipayConfig.FORMAT, alipayConfig.CHARSET,
                alipayConfig.ALIPAY_PUBLIC_KEY,alipayConfig.SIGNTYPE);
        AlipayTradeWapPayRequest alipay_request=new AlipayTradeWapPayRequest();

        // 封装请求支付信息
        AlipayTradeWapPayModel model=new AlipayTradeWapPayModel();
        model.setOutTradeNo(out_trade_no);
        model.setSubject(subject);
        model.setTotalAmount(total_amount);
        model.setTimeoutExpress(timeout_express);
        model.setProductCode(product_code);
        alipay_request.setBizModel(model);
        // 设置异步通知地址
        //alipay_request.setNotifyUrl(alipayConfig.notify_url);
        // 设置同步地址
        alipay_request.setReturnUrl(alipayConfig.return_url);

        // form表单生产

        try {
            // 调用SDK生成表单
            form = client.pageExecute(alipay_request).getBody();

            System.out.println("-------------");
            System.out.println(form);
            System.out.println("---------------------");
        } catch (AlipayApiException e) {
            // TODO Auto-generated catch block
            e.printStackTrace();
        }
        return form;
    }
    //商户订单号   退款原因
    public String refund(String out_trade_no,String refund_reason) {
        String refund_amount =  orderMapper.getTotalPriceByOrdernumber(out_trade_no);
        AlipayClient client = new DefaultAlipayClient(alipayConfig.URL, alipayConfig.APPID,
                alipayConfig.RSA_PRIVATE_KEY, alipayConfig.FORMAT, alipayConfig.CHARSET,
                alipayConfig.ALIPAY_PUBLIC_KEY,alipayConfig.SIGNTYPE);
        AlipayTradeRefundRequest alipay_request = new AlipayTradeRefundRequest();

        AlipayTradeRefundModel model=new AlipayTradeRefundModel();
        model.setOutTradeNo(out_trade_no);
        //model.setTradeNo(trade_no);
        model.setRefundAmount(refund_amount);
        model.setRefundReason(refund_reason);
        //model.setOutRequestNo(out_request_no);
        alipay_request.setBizModel(model);

        AlipayTradeRefundResponse alipay_response = null;
        try {
            alipay_response = client.execute(alipay_request);
        } catch (AlipayApiException e) {
            e.printStackTrace();
        }
        orderMapper.updateOrder(out_trade_no,"5");
        return alipay_response.getBody();
    }
    
    public Boolean updateOrder(String out_trade_no, String isCanceled) {
        return orderMapper.updateOrder(out_trade_no,isCanceled);
    }
}

                              # PayController.java

                              创建一个用于支付的服务接口类。

                              touch touch  src/main/java/com/controller/PayController.java

                              里面提供三个 Rest 接口：

                              1. 支付接口 payment
                              2. 支付成功的回调接口 payresult
                              3. 退款接口 refund

                              具体参考代码如下：

                              @RestController
public class PayController {
    @Autowired
    PayService payService;
    @GetMapping("/payment")
    public String payment(Long userId,Long orderId){
        String form = payService.payment(userId,orderId);
        return form;
    }

    //支付成功回调接口
    @GetMapping("/payresult")
    public String payresult(String out_trade_no){
        //Boolean b = orderService.updateOrder(out_trade_no,"2");
        Boolean b = payService.updateOrder(out_trade_no,"2");
        return b.toString();
    }

    //商户订单号   退款原因
    @GetMapping("/refund")
    public String refund(String out_trade_no, String refund_reason){
        String result =  payService.refund(out_trade_no,refund_reason);
        return result;
    }

}
