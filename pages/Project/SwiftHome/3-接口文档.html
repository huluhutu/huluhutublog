                                                                            # Rest 接口设计

                                                                            # 接口说明

                                                                            # 返回值

                                                                            state:"success" 为成功

                                                                            当 state 的值为 "success" 时，可以通过 data 属性，获取查询的结果集。

                                                                            state:"error" 失败

                                                                            当 state 的值为 "error" 时，可以通过 message 属性，获取错误信息。

                                                                            # 验证

                                                                            最终版的服务接口除了 "/login"、"/regist"、"/show*" 以外，全部开启了验证功能，访问时需要先登录获取 token。在请求需要验证的接口时，需要在请求头上携带 token。

                                                                            案例：

                                                                            let obj = {
  params: { userId: 1, orderId: 28 },
  headers: { authorization: "登录时获取的 token" },
};
axios.get("url", obj).then((response) => {
  //...
});

                                                                            # 添加收货地址

                                                                            说明：

                                                                            根据用户提交的收获地址信息，向数据库 t_address 添加收获地址。

                                                                            URL: /datas/shop/saveAddress

                                                                            请求方法：POST

                                                                            参数：

                                                                            city：String（城市）

                                                                            department：String（小区）

                                                                            houseNumber：Int（门牌号）

                                                                            realName：String （收获人姓名）

                                                                            phone:String（联系电话）

                                                                            userId：Long（所属用户）

                                                                            isdefault：Boolean（是否设置为默认）

                                                                            测试案例：

                                                                            • 测试 URL
                                                                            /datas/shop/saveAddress?city=北京&department=新华开发区&houseNumber=1202&realName=张三&phone=13912323232&userId=1&isdefault=false
                                                                            • 返回 JSON 数据
                                                                            {
    "state": "success"
}
